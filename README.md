# rutas

En la presente practica se hace uso del manejo de rutas con VueJS, se integra la PokeApi 


## Dependencias

- VueJS 3.5.28
- tailwindcss 4.2
- axios 1.13.5




## Instalacion de proyecto

### 1. Clonar repositorio

```sh
git clone https://github.com/licandragon/rutas-web2.git
cd rutas/
npm install
```

### 2. Instalar dependencias
```sh
npm install
```

### 3. Iniciar servidor
```sh
npm run dev
```
